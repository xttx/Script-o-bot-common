  Объявление переменных:
тип_данных имя_переменной;

  Например:
int i;
string strokovaya_peremennaya;
bool perekluchatel;

  В именах переменных можно использовать только буквы, цифры, и знак подчёркивания. При этом переменная не может начинатся с цифры.

  Присвоение значений переменной:
i = 1;
i = 5 + 6;
i = a * 3;
strokovaya_peremennaya = "Я - БОТ.";
perekluchatel = false;

  Абсолюьно нормальная ситуация, когда в присвоении значения переменной, используется она сама:
i = i + 1;     //это прибавет 1 к тому, что уже было в переменной.
i = i * 3;     //а это, умножит переменную на 3.

  Шорткуты.
  Прибавить к переменной единицу можно упрощённым способом:
i++;
  или убавить переменную на один:
i--;
  такая конструкция прибавит к переменной 3:
i += 3;
  а вот это, умножит i на 3:
i *= 3;

  Допускается инициализация значения переменной прямо во время объявления:
int i = 3;
string s = "Кувалдыч, встань в строй!";
bool perekluchatel = false;


  Использование переменных, как параметров к методам и функциям:
int angle = 180;
int distance = 3;
string str = "Hi";
BOT.Rotate(angle);
BOT.Move(distance);
BOT.Say(str);
