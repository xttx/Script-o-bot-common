Статические методы:

Say(string s)       
- говорит текст s.

Move()              
- двигает бота вперёд на 999999 метров.

Move(int n)         
- двигает бота вперёд на n метров.

Rotate(int n)       
- поворачивает бота на n градусов. n может быть отрицательным, для поворота против часовой стрелки.

DestroyContainer()  
- уничтожает контейнер с лутом, на клетке перед ботом.

PutEnergy(int n)    
- выложить n едениц енергии, в коллектор перед ботом.


Статические функции:

int PickUp()                   
- сгребает лут из контейнера перед собой, и возвращает количество сгрёбанного.

int CheckContainerCapacity()   
- просто возвращает количество лута, на клетке перед ботом, не трогая его.
